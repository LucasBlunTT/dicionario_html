DICIONÁRIO DE TAGS HTML

  • <a></a>: Define um hyperlink.
  • <abbr></abbr>: Define uma abreviação.
  • <address></address>: Define informações de contato.
  • <area>: Define uma área em um mapa de imagem.
  • <article></article>: Define um conteúdo independente.
  • <aside></aside>: Define um conteúdo relacionado ao conteúdo principal.
  • <audio></audio>: Define conteúdo de áudio.
  • <b></b>: Define texto em negrito.
  • <base>: Define o URL base para todos os URLs relativos na página.
  • <bdi></bdi>: Isola uma parte do texto que pode ser formatada em uma direção diferente do texto ao redor.
  • <bdo></bdo>: Substitui a direção do texto atual.
  • <blockquote></blockquote>: Define uma seção citada de outra fonte.
  • <body></body>: Define o corpo do documento.
  • <br>: Insere uma quebra de linha.
  • <button></button>: Define um botão clicável.
  • <canvas></canvas>: Usado para desenhar gráficos via script (geralmente JavaScript).
  • <caption></caption>: Define uma legenda para uma tabela.
  • <cite></cite>: Define o título de uma obra.
  • <code></code>: Define um trecho de código.
  • <col>: Especifica propriedades de coluna para cada coluna em um elemento <colgroup>.
  • <colgroup></colgroup>: Especifica um grupo de uma ou mais colunas em uma tabela para formatação.
  • <data></data>: Liga um conteúdo de texto a uma tradução máquina.
  • <datalist></datalist>: Especifica uma lista de opções predefinidas para controles de entrada.
  • <dd></dd>: Define a descrição de um termo em uma lista de definição.
  • <del></del>: Define um texto que foi removido de um documento.
  • <details></details>: Define detalhes adicionais que o usuário pode ver ou ocultar.
  • <dfn></dfn>: Define um termo que está sendo definido.
  • <dialog></dialog>: Define uma caixa de diálogo ou janela.
  • <div></div>: Define uma seção no documento.
  • <dl></dl>: Define uma lista de definição.
  • <dt></dt>: Define um termo em uma lista de definição.
  • <em></em>: Define texto enfatizado.
  • <embed>: Define um contêiner para um recurso externo.
  • <fieldset></fieldset>: Agrupa controles relacionados em um formulário.
  • <figcaption></figcaption>: Define uma legenda para um elemento <figure>.
  • <figure></figure>: Especifica conteúdo independente.
  • <footer></footer>: Define o rodapé de um documento ou seção.
  • <form></form>: Define um formulário HTML para entrada do usuário.
  • <h1> a <h6>: Define cabeçalhos HTML.
  • <head></head>: Contém metadados/informações para o documento.
  • <header></header>: Define o cabeçalho de um documento ou seção.
  • <hr>: Define uma mudança temática no conteúdo.
  • <html></html>: Define a raiz de um documento HTML.
  • <i></i>: Define um texto em itálico.
  • <iframe></iframe>: Define um frame embutido.
  • <img>: Define uma imagem.
  • <input>: Define um controle de entrada.
  • <ins></ins>: Define um texto inserido em um documento.
  • <kbd></kbd>: Define uma entrada de teclado.
  • <label></label>: Define um rótulo para um controle de entrada.
  • <legend></legend>: Define uma legenda para o elemento <fieldset>.
  • <li></li>: Define um item em uma lista.
  • <link>: Define a relação entre um documento e um recurso externo (mais comumente usado para vincular a folhas de estilo).
  • <main></main>: Define o conteúdo principal de um documento.
  • <map></map>: Define um mapa de imagem.
  • <mark></mark>: Define texto marcado/realçado.
  • <meta>: Define metadados sobre um documento HTML.
  • <meter></meter>: Define uma medição escalar dentro de um intervalo conhecido (um medidor).
  • <nav></nav>: Define links de navegação.
  • <noscript></noscript>: Define um conteúdo alternativo para usuários que têm scripts desativados ou um navegador que não suporta script.
  • <object></object>: Define um contêiner para um recurso externo.
  • <ol></ol>: Define uma lista ordenada.
  • <optgroup></optgroup>: Define um grupo de opções relacionadas em um menu suspenso.
  • <option></option>: Define uma opção em um menu suspenso.
  • <output></output>: Define o resultado de um cálculo.
  • <p></p>: Define um parágrafo.
  • <param>: Define um parâmetro para um objeto.
  • <picture></picture>: Define um contêiner para múltiplas fontes de imagem.
  • <pre></pre>: Define texto pré-formatado.
  • <progress></progress>: Representa o progresso de uma tarefa.
  • <q></q>: Define uma citação curta.
  • <rp></rp>: Define o que mostrar em navegadores que não suportam a anotação ruby.
  • <rt></rt>: Define uma explicação/pronúncia de caracteres (usado em anotações ruby).
  • <ruby></ruby>: Define uma anotação ruby (usado em línguas asiáticas).
  • <s></s>: Define texto que não é mais correto.
  • <samp></samp>: Define uma amostra de saída de um programa de computador.
  • <script></script>: Define um script embutido.
  • <section></section>: Define uma seção em um documento.
  • <select></select>: Define um menu suspenso.
  • <small></small>: Define um texto pequeno.
  • <source>: Define uma fonte para o elemento <picture>, <audio>, ou <video>.
  • <span></span>: Define uma seção em um documento.
  • <strong></strong>: Define texto importante.
  • <style></style>: Define informações de estilo para um documento.
  • <sub></sub>: Define texto subescrito.
  • <summary></summary>: Define um título visível para o elemento <details>.
  • <sup></sup>: Define texto sobrescrito.
  • <table></table>: Define uma tabela.
  • <tbody></tbody>: Agrupa o conteúdo do corpo de uma tabela.
  • <td></td>: Define uma célula em uma tabela.
  • <template></template>: Define um conteúdo a ser clonado e inserido posteriormente.
  • <textarea></textarea>: Define uma área de texto multilinha.
  • <tfoot></tfoot>: Agrupa o conteúdo do rodapé de uma tabela.
  • <th></th>: Define uma célula de cabeçalho em uma tabela.
  • <thead></thead>: Agrupa o conteúdo do cabeçalho de uma tabela.
  • <time></time>: Define uma data/horário.
  • <title></title>: Define um título para o documento.
  • <tr></tr>: Define uma linha em uma tabela.
  • <track>: Define faixas de texto para mídia (<video> e <audio>).
  • <u></u>: Define texto sublinhado.
  • <ul></ul>: Define uma lista não ordenada.
  • <var></var>: Define uma variável.
  • <video></video>: Define vídeo embutido.
  • <wbr>: Define uma possível quebra de linha.
